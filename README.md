# 🐍 Snake Game in C++ (SFML/SDL)

This is a graphical Snake game built using **C++** and libraries like **SFML** or **SDL**. The game includes colorful visuals, sound effects, and increasing difficulty levels to make it more exciting and challenging as the player progresses.

## 🎮 Features

- Smooth 2D graphics using SFML or SDL
- Fun sound effects on eating food and collisions
- Increasing speed as the snake grows
- Score display and game over screen
- Real-time keyboard input and movement

## 🛠 Requirements

- C++11 or above
- [SFML](https://www.sfml-dev.org/) or [SDL](https://www.libsdl.org/) installed
- A C++ compiler (g++, clang++, or MSVC)

## 🚀 How to Run

### With SFML:
```bash
g++ -std=c++11 snake.cpp -o snake -lsfml-graphics -lsfml-window -lsfml-system -lsfml-audio
./snake
